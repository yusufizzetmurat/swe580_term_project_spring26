# January 25, 2026

Spent the morning reading about [[GPT]] — specifically the GPT-2 and GPT-3 papers. Important to understand the contrast with [[BERT]] for the related work section: BERT went left in one direction (bidirectional understanding), GPT went right in another (autoregressive generation). The field spent years arguing about which approach was "correct" before realizing they solve different things.

The [[RAG_System]] prototype is taking shape. End-to-end pipeline is working: user query → retrieval → prompt construction → generation. The retrieval step is the weak point — precision is decent but recall on specific factual questions is poor.

Also took time to write proper [[Numpy_Guide]] notes. I use NumPy every day but realized I've been writing some operations in a slow, un-vectorized way. The einsum section was particularly useful — I now understand how to use it for the attention score computation without loops.

## Interesting Observation

GPT-3's in-context learning feels almost magical when you first read about it — give the model a few examples and it figures out the task. But when you think about it mechanically, it makes sense: the attention mechanism can attend to the examples in the prompt and use them as a soft lookup table. The "learning" is pattern matching at inference time.
